## Atalhos Terminal

→ **força uma parada na execução de algo rodando no terminal (ex. Python)**
`Ctrl + C`

→ **copia um texto que está dentro do terminal**
`Ctrl + Shift + C`

→ **cola um texto dentro do terminal**
`Ctrl + Shift + V`

## Atalhos úteis (Windows e Linux)

→ **copia**
`Ctrl + C`

→ **cola**
`Ctrl + V`

→ **recorta**
`Ctrl + X`

→ **desfaz**
`Ctrl + Z`

→ **refaz**
`Ctrl + Shift + Z`
